# EarSaver: YouTube Volume Equalizer

EarSaver is a sleek Chrome extension that automatically normalizes and equalizes YouTube audio volume. It prevents ear-piercing volume spikes and ensures a consistent listening experience across different videos and channels.

![EarSaver Logo](icons/icon128.png)

## 🚀 Features

- **Automatic Normalization**: Dynamically compresses audio to level out volume spikes.
- **Volume Boost**: Boost your audio general level by up to 200%.
- **Seamless Integration**: Works directly in the YouTube player using the Web Audio API.
- **Modern UI**: A beautiful, dark-themed popup with glassmorphism effects.
- **Privacy Focused**: Processes all audio locally on your machine.

## 🛠️ Installation (Development Mode)

If you are not installing from the Chrome Web Store, you can load it manually:

1. Clone or download this repository.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** in the top right corner.
4. Click **Load unpacked** and select the project folder.

## 📖 Usage

1. Click on the EarSaver icon in your browser toolbar.
2. Use the slider to adjust the general volume boost.
3. Audio normalization is active by default on all YouTube videos.

## 💻 Technologies

- **Web Audio API**: For real-time audio manipulation and compression.
- **Chrome Extension API (MV3)**: For browser integration and storage.
- **Vanilla JS, HTML, CSS**: Lightweight and fast.
